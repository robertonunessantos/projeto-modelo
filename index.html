Sobre

Por que usar este modelo?

Este site serve como ponto de partida para projetos escolares, atividades práticas, introdução ao desenvolvimento web e publicação de páginas no GitHub Pages.

O código utiliza nomes de classes, ids e funções em português para facilitar a leitura e o entendimento de estudantes iniciantes.
