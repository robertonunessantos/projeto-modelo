# projeto-modelo
Projeto modelo para professores e alunos

Projeto simples para aulas introdutórias de desenvolvimento web.

## Estrutura

```text
projeto-modelo/
├── index.html
├── css/
│   └── style.css
└── js/
    └── script.js
```

## Objetivo

Este projeto serve como modelo para professores de programação trabalharem:

- estrutura básica de uma página HTML;
- organização visual com CSS;
- interação simples com JavaScript;
- uso de pastas separadas para cada tipo de arquivo;
- publicação de páginas no GitHub Pages.

## Como usar

1. Abra a pasta `projeto-modelo` no VS Code.
2. Abra o arquivo `index.html` no navegador.
3. Edite os arquivos `style.css` e `script.js` para personalizar o projeto.
4. Publique no GitHub Pages se desejar transformar em site online.
